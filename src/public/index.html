Pennant

I recently created Pennant, a real-time collaborative notebook that integrates note-taking and code execution into a web-based tool. It uses conflict-free replicated data types (CRDTs) to broadcast low-latency document updates to all connected clients. Its execution engine also safely runs untrusted user code using language sandboxing and hardened Docker configurations.
